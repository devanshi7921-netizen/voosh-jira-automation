# Voosh Jira Automation Project

## Objective
Demonstrate automated workflows connecting Gmail customer requests to actionable Jira tickets for multiple stakeholders.

## Scenario
Customer emails need to be converted into Jira tickets automatically, split into sub-tasks for frontend, backend, QA teams, with tracking of delays and overall progress.
